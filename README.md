# Tavant-NewRez

UI Developer

Location: Coppell, TX-5 Days work from office

Tavant

Job Description:
We are seeking a talented UI Developer to join our dynamic team. The ideal candidate will be responsible for designing and implementing high-quality user interfaces for our applications. You will work closely with UX designers and back-end developers to create seamless and efficient user experiences. The role involves developing responsive web applications, writing clean and maintainable code, and ensuring optimal performance across various devices and browsers.

Key Responsibilities:

- Develop and maintain user interface components and frameworks.
- Collaborate with UX/UI designers to transform wireframes and mockups into functional user interfaces.
- Write clean, maintainable, and efficient code in accordance with industry best practices.
- Optimize applications for maximum speed and scalability.
- Ensure the technical feasibility of UI/UX designs.
- Perform regular maintenance, updates, and troubleshooting of existing applications.
- Participate in code reviews and contribute to team knowledge sharing.
- Stay updated with emerging trends and technologies in front-end development.

Skills Required:

- Proficiency in HTML, CSS, and JavaScript.
- Experience with front-end frameworks such as React, Angular, or Vue.js.
- Knowledge of responsive design principles and frameworks (Bootstrap, Foundation).
- Familiarity with version control systems, preferably Git.
- Understanding of web accessibility standards (WCAG).
- Ability to write clean, reusable code following best practices.
- Strong problem-solving skills and attention to detail.
- Excellent communication and collaboration abilities.

Tools and Technologies:

- Front-end frameworks (React, Angular, or Vue.js).
- CSS preprocessors (Sass, LESS).
- Version control systems (Git).
- Build tools (Webpack, Gulp, or Grunt).
- IDEs/Text Editors (VS Code, Sublime Text, or similar).
- Browser developer tools for debugging and performance testing.
- Familiarity with RESTful APIs and integration with back-end services.
